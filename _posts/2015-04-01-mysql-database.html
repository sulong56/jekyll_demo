一、数据库

1.基础

数据库(Database,DB):按一定结构存储数据的仓库。

2.登录/退出MySQL

mysql -u用户名 -p密码 -h服务器名称 -D数据库名称 -P端口

mysql>exit; 或 mysql>quit; 或 mysql>\q;

说明:MySQL命令必须以分号结尾

3.常用命令

当前MySQL版本

mysql>SELECT version();

查看当前用户

mysql>SELECT user();

查看当前日期时间

SELECT now();

查看当前的日期

SELECT CURDATE();

查看当前的时间

SELECT CURTIME();

向日期添加指定的时间间隔

DATE_ADD(date,INTERVAL expr type)

说明:date 参数是合法的日期表达式。expr 参数是您希望添加的时间间隔。type 参数可以是下列值：YEAR(年) DAY(日) HOUR(小时) MINUTE(分钟) SECOND(秒) MICROSECOND(微秒) ...

查找中使用了: SELECT OrderId,DATE_ADD(OrderDate,INTERVAL 2 DAY) AS OrderPayDate FROM Orders

从日期减去指定的时间间隔

DATE_SUB(date,INTERVAL expr type)

说明:date 参数是合法的日期表达式。expr 参数是您希望添加的时间间隔。type 参数可以是下列值：YEAR(年) DAY(日) HOUR(小时) MINUTE(分钟) SECOND(秒) MICROSECOND(微秒) ...

查找中使用了: SELECT OrderId,DATE_SUB(OrderDate,INTERVAL 2 DAY) AS OrderPayDate FROM Orders

返回两个日期之间的天数

DATEDIFF(date1,date2)

说明:date1 和 date2 参数是合法的日期或日期/时间表达式。

查找中使用了: SELECT DATEDIFF('2008-12-30','2008-12-29') AS DiffDate

查看MySQL当前用户占用的连接数

show processlist;

MYSQL对数据库操作

查看当前数据库

SELECT database();

查看当前服务器下所有的数据库列表

show databases;

创建数据库

create database [if not exists] 数据库名称 [[default] character set '编码方式']

查看数据库的创建命令

show create database 数据库名称

修改数据库编码方式

alter database 数据库名称 [default] character set '编码方式'

删除数据库

drop database [if exists] 数据库名称

打开数据库

use 数据库名称

导出数据库

mysqldump --databases db1 db2 > db1.db2.sql

说明:将数据库dbl和db2备份到db1.db2.sql文件，db1.db2.sql是一个文本文件，文件名任取。

导出表

mysqldump --opt school abc > school.sql

说明:将数据库school中的abc表备份到school.sql文件，school.sql是一个文本文件，文件名任取。

导入数据库，数据表

use 数据库名称;

SOURCE 之前导出的数据库文件完全路径;
