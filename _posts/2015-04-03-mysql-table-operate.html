2.SELECT -- 查找记录

 SELECT 表达式[,....] from 数据表参照

  [where  条件]

  [group by 分组字段]

  [having 分组条件]

  [order by 排序字段[asc|desc][,...]]

  [limit [偏移值,]记录数目]

说明:

A、星号(*)表示所有字段

B、字段 [[as]别名]

C、数据表 [[as] 别名]

D、order by,对于查找结果进行排序，其中asc(默认)为升序，desc为降序。

E、limit,限制返回的记录数,偏移值从0开始编号。

F、分页时的偏移值的计算公式：(页码-1)*每页显示的记录数

where条件

A.[not] between ... and

字段名称 [not] between ... and ...

B.[not] in ...

字段名称 [not] in(值,....)

C.[not] like '表达式'

字段名称 [not] like '表达式'

D、逻辑运算符: not、and、or
